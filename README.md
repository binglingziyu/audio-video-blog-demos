# 运行

## 方式一

CLion -> Get From VCS

## 方式二

```shell script
> git clone git@github.com:binglingziyu/audio-video-blog-demos.git
> cd audio-video-blog-demos
> git submodule init
> git submodule update

Open with CLion
```
